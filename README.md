
emq_gps
=========

The plugin adds GPS 0.1 protocol support to the emqttd broker.

The GPS clients could pub/sub with the MQTT topics, and talk with the MQTT clients!

Configure the Plugin
--------------------

etc/emq_gps.conf

Load the Plugin
---------------

./bin/emqttd_ctl plugins load emq_gps
